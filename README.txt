== Installation ==

**From your WordPress dashboard**
1. Visit 'Plugin > Add new'.
2. Search for 'WikiLMS'.
3. Activate WikiLMS from your Plugins page.

**From WordPress.org**
1. Search, select and download WikiLMS.
2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.

== Frequently Asked Questions ==

= What is WikiLMS? = 

Transform your WordPress website into an interactive online learning hub with WikiLMS, an intuitive Learning Management System (LMS) plugin.

= Is this plugin free? =

This plugin is completely free!  But to make it work, you need to install the WikiLMS plugin. Even the templates included in this plugin are free as well. 

= Can I use WikiLMS with any WordPress site? = 

Yes, WikiLMS seamlessly integrates with any WordPress site, making it easy to use with your existing website.

= Can I use WikiLMS with Elementor? =

Yes, with WikiLMS, you can integrate with Elementor page builder.

= How do I get a form shortcode? =

You can use the Contact Form 7 shortcode for this
